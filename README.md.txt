**Real Estate Management System**

The real estate management system is a web application designed for any real estate agency, which has an intuitive and effortless to use to interface for both its agency owner and its customers. Here, owners and customers have to sign-up in order to access the website. The owner can post the property that needs to be rented in the property listings. Here, the customers can browse through various properties listed on the website which will enable the customer to find the property as per his/her requirements.

This application is hosted locally using XAMPP and developed with:
1.PHP - backend scripts
2.MySQL - database design
3.HTMl,CSS and Bootstrap - frontend

Create a database named "realestate", import the provided sql file and you're good to go.
